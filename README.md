## よくある質問

- どんなエンジニアになりたいか(30 代以上のビジョンを踏まえて)  
  ある程度、歳を重ねたらマネジメントと上流工程の仕事をしっかりやっていきたいです。

- 一番好きな言語は何ですか  
  強いていうならば Python3 ですが、ひとつの技術にこだわることはないです。

- 知らない人と話すことに抵抗はありませんか  
  No.自分の知らない何か持っているなと思う思考回路なので怖い人でなければ、積極的に話しかけます。

- 新しいことを学んでいくことに苦はないか  
  いま持っている技術で何年もやっていけるとは 1 ミリも思っていないので技術者たるもの技術を求めるべき

- 制作した成果物を教えてください  
  現在開発中です。  
  会社で作成したモノについては社内から出せない為、似たようなモノを制作してリリースするよう検討中です。

- 今後、どんなエンジニア目指して活躍したいか  
  AWS を含むクラウドを主軸にアプリケーション開発ができるバックエンドエンジニアとして活躍したいです。

- マネジメントと開発、インフラ運用保守などいくつかやっているがマネジメントを担当するだけとなった場合は対応できるか  
  結論を言えばできます。理由を述べれば、私は誰かの憧れの対象になりたいということが挙げられるかと思います。  
  そういったことの背景には私自身が結構なワンマンアーミーで一人で運用保守・開発をやってきた身であり  
  当時の自分の目の前に積極的にリードしてくれる強いリーダがいたらきっと憧れてついていくと思うからです。  
  また、私的な面で言えば私は技術も好きですけど技術が好きな技術者も好きです。  
  ですのでそういった技術好きな技術者が迷うことなく環境で活躍できるようにリードするということにはむしろ、興味があります。

- PM/PL といったリーダーという立場はどうあるべきか  
  端的に言えば、「矢面に立つ」ということです。迷うことなく恐れずに誰よりも先にはじめの一歩を踏み出すことです。
